# LLM Failure Classification Framework

This framework categorizes AI model vulnerabilities discovered during adversarial testing.

## 1. Prompt Injection
User input overrides system instructions.

## 2. Jailbreak
Prompt bypasses alignment safeguards.

## 3. Unsafe Content Generation
Model produces harmful or restricted content.

## 4. Hallucination
Model fabricates false information.

## 5. Data Leakage
Model exposes sensitive or private data.

## 6. Instruction Override
Model ignores system policies.
